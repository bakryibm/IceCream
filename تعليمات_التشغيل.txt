========================================
    نظام إدارة محل الآيس كريم
========================================

تم إنشاء برنامج متكامل لإدارة محل بيع الآيس كريم بنجاح!

🎯 المميزات المتوفرة:
✅ نظام نقطة البيع مع واجهة عربية
✅ إدارة المنتجات والمخزون
✅ إدارة العملاء والمستخدمين
✅ نظام التقارير (PDF و Excel)
✅ قاعدة بيانات SQLite محلية
✅ واجهة Material Design حديثة
✅ دعم كامل للغة العربية

📋 طرق التشغيل:

الطريقة الأولى - التشغيل المباشر:
1. افتح Command Prompt أو PowerShell
2. انتقل إلى مجلد المشروع
3. اكتب: dotnet run --project IceCreamShop.WPF

الطريقة الثانية - استخدام ملف التشغيل:
1. انقر نقراً مزدوجاً على ملف run.bat

الطريقة الثالثة - بناء ملف تنفيذي:
1. انقر نقراً مزدوجاً على ملف build.bat
2. ستجد الملف التنفيذي في مجلد Release

🔐 بيانات تسجيل الدخول الافتراضية:
اسم المستخدم: admin
كلمة المرور: admin123

📁 ملفات المشروع:
- IceCreamShop.Core: النماذج والواجهات الأساسية
- IceCreamShop.Data: طبقة البيانات وقاعدة البيانات
- IceCreamShop.WPF: واجهة المستخدم
- icecreamshop.db: قاعدة البيانات (تُنشأ تلقائياً)

🛠️ التقنيات المستخدمة:
- C# .NET 6
- WPF (Windows Presentation Foundation)
- Entity Framework Core
- SQLite Database
- Material Design UI
- iTextSharp (PDF)
- EPPlus (Excel)

📊 الوظائف المتاحة:

1. نقطة البيع:
   - إضافة المنتجات للسلة
   - حساب الإجمالي والضرائب
   - طرق دفع متعددة
   - طباعة الفواتير

2. إدارة المنتجات:
   - إضافة منتجات جديدة
   - تعديل الأسعار والكميات
   - تصنيف حسب الفئات والنكهات

3. إدارة المخزون:
   - تتبع الكميات
   - تنبيهات المخزون المنخفض
   - تسجيل حركات المخزون

4. التقارير:
   - تقارير المبيعات اليومية والشهرية
   - تقارير المخزون
   - أكثر المنتجات مبيعاً
   - تصدير PDF و Excel

5. إدارة المستخدمين:
   - إضافة مستخدمين جدد
   - صلاحيات مختلفة (مدير، بائع)
   - تتبع نشاط المستخدمين

🔧 متطلبات النظام:
- Windows 10 أو أحدث
- .NET 6.0 Runtime
- 100 MB مساحة فارغة
- 2 GB ذاكرة وصول عشوائي

⚠️ ملاحظات مهمة:
- قاعدة البيانات تُنشأ تلقائياً عند أول تشغيل
- لعمل نسخة احتياطية، انسخ ملف icecreamshop.db
- البرنامج يدعم اللغة العربية بالكامل
- جميع التقارير باللغة العربية

🆘 حل المشاكل الشائعة:

مشكلة: لا يعمل البرنامج
الحل: تأكد من تثبيت .NET 6.0 Runtime

مشكلة: خطأ في قاعدة البيانات
الحل: احذف ملف icecreamshop.db وأعد التشغيل

مشكلة: لا تظهر الخطوط العربية
الحل: تأكد من تثبيت خطوط اللغة العربية في Windows
