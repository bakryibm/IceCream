🚀 HOW TO START (كيفية البدء):
انقر نقراً مزدوجاً على: main-app.bat
Double-click: main-app.bat

========================================
💰 CURRENCY (العملة):
All prices are in Omani Rial (ر.ع)
جميع الأسعار بالريال العُماني (ر.ع)

========================================
🏪 MAIN FEATURES (الميزات الرئيسية):

1. 🛒 SALES POINT (نقطة البيع):
   - Click "نقطة البيع" or "بدء البيع"
   - Select products to add to cart
   - View total in Omani Rial
   - Complete sales transactions
   - Clear cart when needed

2. 📦 PRODUCT MANAGEMENT (ادارة المنتجات):
   - Click "إدارة المنتجات"
   - Add new products with name, price, category
   - Edit existing products
   - Delete products
   - View all products in organized list

3. 📊 INVENTORY MANAGEMENT (ادارة المخزون):
   - Click "إدارة المخزون"
   - View current stock levels
   - Update quantities
   - Set minimum stock levels
   - Get low stock alerts
   - View inventory statistics

4. 📈 REPORTS SYSTEM (نظام التقارير):
   - Click "التقارير"
   - Generate Sales Reports
   - Generate Inventory Reports
   - Generate Financial Reports
   - View detailed statistics
   - Print/Export options

5. 📊 SALES HISTORY (تاريخ المبيعات):
   - Click "تاريخ المبيعات"
   - View all past transactions
   - Filter by date range
   - Search specific sales
   - View refunds and returns
   - Export sales data

6. 👥 CUSTOMER MANAGEMENT (ادارة العملاء):
   - Click "ادارة العملاء"
   - Add new customers
   - Manage membership types
   - Track purchase history
   - Customer statistics
   - Export customer data

7. 👤 USER MANAGEMENT (ادارة المستخدمين):
   - Click "ادارة المستخدمين"
   - Add/edit system users
   - Assign roles and permissions
   - Monitor user activity
   - Security settings
   - Access control

8. ⚙️ SYSTEM SETTINGS (اعدادات النظام):
   - Click "الاعدادات"
   - Shop information setup
   - Currency and language settings
   - Backup and restore
   - Receipt configuration
   - Security preferences

========================================
🎯 QUICK START GUIDE (دليل البدء السريع):

STEP 1: Start Application
شغل البرنامج باستخدام main-app.bat

STEP 2: Explore Main Window
استكشف النافذة الرئيسية والقوائم

STEP 3: Try Sales
جرب نقطة البيع - اضغط "بدء البيع"

STEP 4: Add Products
أضف منتجات جديدة من "إدارة المنتجات"

STEP 5: Check Inventory
تحقق من المخزون في "إدارة المخزون"

STEP 6: View Reports
اعرض التقارير من "التقارير"

========================================
🔧 TROUBLESHOOTING (حل المشاكل):

PROBLEM: Application won't start
SOLUTION: Make sure .NET 6.0 is installed

PROBLEM: Windows shows security warning
SOLUTION: Click "More info" then "Run anyway"

PROBLEM: Arabic text not displaying
SOLUTION: Make sure Windows Arabic support is enabled

PROBLEM: Application crashes
SOLUTION: Check CURRENT_STATUS.txt for known issues
